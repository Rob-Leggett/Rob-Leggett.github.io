## Content

Blog posts are written in Markdown/MDX and stored under content/publish/.

Each post includes standard frontmatter fields like:
```yaml
---
title: "Post title"
date: "2025-01-10"
slug: "post-slug"
feature_image: "/blog/post/feature-image.png"
---
```
