## 📋 Key Formatting Requirements (At a Glance)

### MUST DO:

1. Title: "Deep Dive - [Topic]"
2. Opening: "Welcome to the deep dive series!"
3. Frontmatter: Include all 5 sections (title, meta, og, twitter, tags)
4. Tags: 9 items with "Robert Leggett" LAST
5. Before You Start: Exactly 3 subsections
6. H2 Sections: End each with `---` (horizontal rule)
7. Code Blocks: Include language identifier (hcl, yaml, python, etc.)
8. Images: Include alt text AND caption
9. Terminology Section: "Okay, what terminology do I need to understand?"
10. Closing: "Finally, we made it, did this help?"

### Article Metrics:

- Reading Time: 10-20 minutes
- H2 Sections: 15-20+
- H3 Subsections: 20-40+
- Code Examples: 5-15+
- Images: 3-8+
- Total Lines: 700-900+
- Terminology Terms: 10-15
